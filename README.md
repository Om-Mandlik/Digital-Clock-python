#  PyQt5 Digital Clock

A simple and elegant digital clock application built using **PyQt5**. This desktop app displays the current time in real-time with a clean and minimal interface.

---

## Features

-  Real-time clock updating every second  
-  Stylish UI with custom fonts and colors  
-  Lightweight desktop application  
-  Easy to run and modify  


###  Prerequisites

- Python 3.x installed on your system  


**How It Works**
- Uses QTimer to trigger updates every second
- Fetches current time using QTime
- Displays time using a QLabel
- Applies styling with Qt Stylesheets
